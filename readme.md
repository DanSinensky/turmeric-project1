# The Aggregate Press Informer

A news application that creates customizable front pages by topic/date/reporter/outlet, saves them in localstorage, and links to the full articles for later reading

### Technologies Used

- HTML
- CSS
- JavaScript
- jQuery
- Google Fonts
- API
  - News API
